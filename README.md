# ewelina-ai-portfolio

Ready-to-deploy React + Vite project for Ewelina's AI Fashion Portfolio.

## Quick guide (idiot-proof) — deploy to Vercel via GitHub (no terminal)

1. Download and unzip this folder.
2. Create a new GitHub repository (any name).
3. Upload all files from the unzipped folder onto the repository using **Add file → Upload files**.
4. Sign in to https://vercel.com and choose **New Project → Import Git Repository** and pick your repo.
5. Vercel will detect a Vite project. Deploy. After a minute your site will be live.

## Edit content
- Replace images in `public/images/` with your files.
- Edit `src/App.jsx` to change texts (Your Name, email, prompts).
- For video/audio links, set public URLs in the AIDEAS case study section.

## Local development (optional)
If you prefer to run locally:
- Install Node.js (LTS)
- In project root run:
  ```
  npm install
  npm run dev
  ```
- Open shown local URL (usually http://localhost:5173)
